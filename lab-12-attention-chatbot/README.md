# Lab 12 – Generative Model with Attention

## Objective

To implement a text generation model using attention for better contextual understanding.

---

## About the Lab

In this lab, we built a simple chatbot that generates responses based on user input.
We used an LSTM model along with an attention mechanism to focus on important words in the input sentence.

---

## What I Did

* Preprocessed text data (tokenization and vocabulary creation)
* Built an LSTM encoder model
* Added an attention mechanism to focus on relevant words
* Generated responses using the trained model

---

## Output

Example:
Input: how are you
Output: i am fine

---

## Key Learning

* Attention helps the model focus on important words
* Improves response quality compared to basic LSTM

---

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
